SimSiam¶

solo.losses.simsiam.simsiam_loss_func(p: torch.Tensor, z: torch.Tensor, simplified: bool = True) → torch.Tensor[source]

Computes SimSiam’s loss given batch of predicted features p from view 1 and a batch of projected features z from view 2.

Parameters

  • p (torch.Tensor) – Tensor containing predicted features from view 1.

  • z (torch.Tensor) – Tensor containing projected features from view 2.

  • simplified (bool) – faster computation, but with same result.

Returns

SimSiam loss.

Return type

torch.Tensor

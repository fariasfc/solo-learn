ReSSL¶

solo.losses.ressl.ressl_loss_func(q: torch.Tensor, k: torch.Tensor, queue: torch.Tensor, temperature_q: float = 0.1, temperature_k: float = 0.04) → torch.Tensor[source]

Computes ReSSL’s loss given a batch of queries from view 1, a batch of keys from view 2 and a queue of past elements.

Parameters

  • query (torch.Tensor) – NxD Tensor containing the queries from view 1.

  • key (torch.Tensor) – NxD Tensor containing the queries from view 2.

  • queue (torch.Tensor) – a queue of negative samples for the contrastive loss.

  • temperature_q (float, optional) – [description]. temperature of the softmax for the query. Defaults to 0.1.

  • temperature_k (float, optional) – [description]. temperature of the softmax for the key. Defaults to 0.04.

Returns

ReSSL loss.

Return type

torch.Tensor

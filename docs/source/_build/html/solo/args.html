setup¶

parse_args_pretrain¶

solo.args.setup.parse_args_pretrain() → argparse.Namespace[source]

Parses dataset, augmentation, pytorch lightning, model specific and additional args.

First adds shared args such as dataset, augmentation and pytorch lightning args, then pulls the model name from the command and proceeds to add model specific args from the desired class. If wandb is enabled, it adds checkpointer args. Finally, adds additional non-user given parameters.

Returns

a namespace containing all args needed for pretraining.

Return type

argparse.Namespace

utils¶

parse_args_pretrain¶

solo.args.utils.additional_setup_pretrain(args: argparse.Namespace)[source]

Provides final setup for pretraining to non-user given parameters by changing args.

Parsers arguments to extract the number of classes of a dataset, create transformations kwargs, correctly parse gpus, identify if a cifar dataset is being used and adjust the lr.

Parameters

  • args (Namespace) – object that needs to contain, at least:

  • dataset (-) – dataset name.

  • brightness (-) – required augmentations settings.

  • contrast – required augmentations settings.

  • saturation – required augmentations settings.

  • hue – required augmentations settings.

  • min_scale – required augmentations settings.

  • multicrop (-) – flag to use multicrop.

  • dali (-) – flag to use dali.

  • optimizer (-) – optimizer name being used.

  • gpus (-) – list of gpus to use.

  • lr (-) – learning rate.

  • [optional] –

  • gaussian_prob (-) – optional augmentations settings.

  • solarization_prob – optional augmentations settings.
